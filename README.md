## 📌 About The Project

Learning Style Prediction System is a Machine Learning project developed to identify a student's preferred learning style using the VARK framework.

The system analyzes questionnaire responses and learning behavior patterns to classify students into Visual, Auditory, Reading/Writing, or Kinesthetic learning categories.

The project helps educators and learners understand individual learning preferences and supports personalized learning strategies.
